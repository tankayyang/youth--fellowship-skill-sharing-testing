Basic Elements of a Web Page

Understanding the anatomy of a website is the first step in mastering web development. Let's break down the common components found on modern sites.

The Building Blocks of the Web

Every website you visit is constructed using a series of specialized sections. By using semantic HTML5 tags, developers can tell both the browser and search engines what each part of the page represents.

1. Header & Navigation

The header usually contains the identity of the site (logo/title) and the navigation bar. The navigation bar provides links to help users move through the website efficiently.

Why it matters: A good header improves user experience by helping visitors find pages quickly.

2. Semantic Structure

Using tags like <main>, <article>, and <section> helps assistive technologies like screen readers understand the hierarchy of your information.

<main>
  <article>Our Main Story</article>
  <aside>Related Links</aside>
</main>

Semantic HTML makes your site easier to read, maintain, and improve for SEO and accessibility.

3. Buttons, Forms & Footer

Buttons encourage action, forms collect input, and the footer gives extra links and useful information.

  • Buttons can help guide users to click important actions.
  • Forms collect things like names, emails, and messages.
  • Footers often contain contact info, policies, and social links.
